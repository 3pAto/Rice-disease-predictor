# Rice-disease-predictor
Advanced Rice Disease Detection using 10 Machine Learning Models.

## Features
- 🔬 Detects 4 rice diseases: Bacterial Blight, Blast, Brown Spot, False Smut
- 🤖 10 ML models: Random Forest, SVM, KNN, XGBoost, LightGBM, etc.
- 🌐 Upload images or use URL
- 📊 Model comparison & agreement visualization
- 💡 Disease management recommendations
